## What?

Suppose we have table, `users` and `posts` with structure:

```ruby
User(
  id: integer,
  name: string,
  email: string,
)
```

and

```ruby
POST(
  id: integer,
  title: string,
  article: varchar,
  user_id: integer,
)
```

### Create

POST /posts

```JSON
{
  "data": {
    "type": "posts",
    "attributes": {
      "title": "test_title",
      "article": "test article"
    },
    "relationships": {
      "user": {
        "data": {
          "type": "users",
          "id": 2
        }
      }
    }
  }
}
```

and the response from server:

```JSON
{
  "data": {
    "type": "posts",
    "id": 442,
    "attributes": {
      "title": "test_title",
      "article": "test article",
    },
    "links": {
      "self": "https://test.example.com/posts/442"
    }
  }
}
```
